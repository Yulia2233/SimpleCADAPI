## API Definition

```python
def scale_rscalarfield(field: ScalarField, factors: Tuple[float, float, float]) -> ScalarField
```

## Description

Scale a scalar field around the origin.

## Parameters

### field

- **Description**: Input scalar field.

### factors

- **Description**: Scale factors `(sx, sy, sz)`.

## Returns

ScalarField: Scaled scalar field.
